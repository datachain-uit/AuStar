# AuStar: Improving Vitamin D Deficiency Prediction via SMOTE-Based Data Augmentation and Stacking Ensemble Model in Clinical Medicine

AuStar is a machine learning–based research project focusing on **vitamin D deficiency prediction** using clinical and demographic data.  
The proposed framework integrates **SMOTE-based data augmentation** with a **stacking ensemble learning strategy** to improve predictive performance on imbalanced healthcare datasets.

This repository accompanies an academic study and is intended for **research, reproducibility, and benchmarking purposes**.

---

## 📌 Key Contributions

- Comprehensive preprocessing pipeline for clinical tabular data  
- Handling class imbalance using **SMOTE-based data augmentation**
- Construction of a **stacking ensemble model** combining multiple base learners
- Extensive evaluation using clinically relevant metrics:
  - Accuracy (ACC)
  - Balanced Accuracy
  - F1-score (macro & weighted)
  - Matthews Correlation Coefficient (MCC)
  - ROC-AUC and PR-AUC
- Empirical validation on **NHANES survey datasets**

---

## 📊 Dataset

- **Source**: National Health and Nutrition Examination Survey (NHANES)
- **Data type**: Clinical, biochemical, and demographic variables
- **Target**: Vitamin D deficiency classification
- **Label imbalance**: Addressed using SMOTE variants

> ⚠️ Due to NHANES data usage policies, raw data are **not redistributed** in this repository.  
> Users should obtain the data directly from the official NHANES website and follow the preprocessing instructions below.
