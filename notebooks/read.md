# Notebooks

This directory is dedicated to the analytical code and experimental evaluations conducted during the research phase of the Vitamin D deficiency prediction project.

## 1. Exploratory Data Analysis (EDA)

Analytical scripts used to investigate the health clinic dataset:

- Statistical profiling of clinical parameters and patient records.
- Correlation analysis between biochemical markers and Vitamin D levels.
- Data distribution assessments to identify patterns in medical features.

## 2. Experimental Result Analysis

Code for analyzing the performance of machine learning experiments:

- Evaluation of data enrichment outcomes using SVM-SMOTE and stacking algorithms.
- Comparative analysis of predictive accuracy across different ensemble learning configurations.
- Metrics calculation including confusion matrices, precision-recall, and F1-scores.

## 3. Data Visualization

Scripts for generating research-grade visualizations:

- Implementation of plotting logic using Matplotlib for research performance visualization.
- Graphic representation of feature importance and model convergence rates.
- Visual comparison of class distribution before and after applying balancing techniques.
